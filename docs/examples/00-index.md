# Examples

Patterns validated against the live SISP/Vinti4 gateway with a standalone consumer app. Each example is self-contained and uses only the package's public API. Examples are grouped by HTTP adapter; inside each adapter you get the storage variants (knex, Prisma) plus the frontend and behavior patterns.
